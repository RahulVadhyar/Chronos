Detailed Description

A class to represent a polygon object.

A polygon object is a shape that can have any number of vertices and can be textured These vertices can also be updated on the fly after creation. This is done by creating a vertex buffer and copying the vertices to it The indices are generated using the earcut library. This library takes a list of vertices and generates a list of indices that represent the triangles that make up the polygon It uses a polygon triagulation algorithm to do this. Since the vertices can be updated any time, we have to create multiple vertex buffers and update them as the frames are being updated.
